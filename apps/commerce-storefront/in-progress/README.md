# In Progress

This directory holds active feature work managed by the agentic pipeline.

Files created automatically by the orchestrator:

| File | Purpose |
|---|---|
| `<slug>_SPEC.md` | Feature specification (human-authored) |
| `<slug>_STATE.json` | Machine-readable pipeline state (auto-managed) |
| `<slug>_TRANS.md` | Human-readable transition log (auto-generated) |
| `<slug>_CHANGES.json` | Change manifest (generated by docs-archived agent) |

> Do not edit `_STATE.json` or `_TRANS.md` manually. Use `npm run pipeline:*` commands.
